# Neutrino oscillation phases

Manim scene that shows two mass-eigenstate phases and the resulting
oscillation probabilities.

---

## Files

- `neu_osc_phases.py`
  Manim scene. Reads parameters from `run.cfg`.

- `run.cfg`
  16:9 hires configuration and scene/UI parameters.

---

## How to run

```bash
manim -pqh neu_osc_phases.py NeutrinoOscillationPhases -o output_name
```
