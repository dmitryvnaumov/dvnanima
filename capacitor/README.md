# Capacitor

Manim scene that illustrates an AC-driven capacitor with oscillating charge
and current in a closed loop.

---

## Files

- `Capacitor.py`
  Manim scene. Reads parameters from `run.cfg`.

- `run.cfg`
  16:9 hires configuration and scene parameters.

---

## How to run

```bash
manim -pqh Capacitor.py Capacitor -o output_name
```
