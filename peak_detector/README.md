# Peak detector

Manim scene that visualizes a diode peak detector with a charging capacitor.

---

## Files

- `peak_detector.py`
  Manim scene. Reads parameters from `run.cfg`.

- `run.cfg`
  16:9 hires configuration and scene parameters.

---

## How to run

```bash
manim -pqh peak_detector.py PeakDetector -o output_name
```
