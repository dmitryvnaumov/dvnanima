# Neutrino mass hierarchy

Manim scene that visualizes a mass-hierarchy balance with two oscillating
mass eigenstates.

---

## Files

- `balance.py`
  Manim scene. Reads parameters from `run.cfg`.

- `run.cfg`
  16:9 hires configuration and scene parameters.

---

## How to run

```bash
manim -pqh balance.py NeutrinoHierarchyScales -o output_name
```
